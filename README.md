## Getting Started

First, run the development server:

```bash
npm install
# then
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) will be the base url for the server(nextjs app).

You can start editing the api files by modifying `app/api/chat/route.ts`. The page auto-updates as you edit the file.
